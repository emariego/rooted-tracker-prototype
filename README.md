# rooted-tracker-prototype
rapid-prototype digital wellness application built with Python and Streamlit, utilizing AI-augmented development to track 'Rooted &amp; Resilient' daily devotionals.

Rooted & Resilient: Devotional Tracker PrototypeProject OverviewThis application is a functional prototype designed to transition a hand-drawn UX concept for a 7-day spiritual reset into a live technical tool. It was developed to demonstrate AI-augmented rapid prototyping—moving from a wireframe to a deployed Cloud application in under 60 minutes.Technical StackLanguage: Python Framework: Streamlit (Frontend/State Management)Development Workflow: AI-Augmented "Vibe Coding" via Claude Code / LLM Orchestration Deployment: Streamlit CloudKey FeaturesDynamic State Management: Uses st.session_state to track task completion across a 7-day lifecycle.SVG Progress Visualization: A real-time, clockwise progress circle that calculates completion percentages for the active day.Accordion UI Components: Modular sections for the "Rooted Word," "Prayer Lab," and "Community Seed" to minimize cognitive load and screen time.The "Technologist" InsightThis project applies the same procedural automation logic I utilized at studios like ILM and MPC . Just as I engineered rigs and simulations to automate complex VFX workflows, this tool abstracts technical complexity into an intuitive, artist-friendly interface for digital wellness.
