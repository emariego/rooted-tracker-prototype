# 🌿 Rooted & Resilient: Devotional Tracker Prototype

### **[🚀 Live Demo Link](https://rooted-tracker-prototype-v001.streamlit.app/)**

## **Overview**
This project is a high-speed technical prototype designed to bridge the gap between a **UX wireframe** and a **functional digital tool**. Developed as part of my portfolio for the **BBC AI Creative Lab**, it demonstrates the power of **AI-augmented rapid prototyping** in creating "artist-friendly" tools for digital wellness.

## **Technical Breakdown**
* **Language:** Python.
* **Framework:** Streamlit for state management and UI .
* **Architecture:** Utilizes `st.session_state` to maintain task persistence across a 7-day cycle.
* **UI/UX:** Features a dynamic SVG-based circular progress bar that updates in real-time via user interaction with accordions.

## **The "Creative Technologist" Connection**

* Just as I engineered complex rigs for *Jurassic World* to automate Character Effects (CFX), this tool automates the "Life Admin" of a spiritual reset.
